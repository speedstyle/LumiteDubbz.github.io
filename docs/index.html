Official Support Server

SauceBot is a primarily administrative Discord bot, made with Discord.js. It is designed to be similar to the DEA bot, so as to make users familier with the commands before they've even used SauceBot. Most of the commands have the same names.

It is designed to be like DEA because lots of servers I am a part of use DEA, becuase it's creator is an ex-YouTuber of the game.
